# ArtConnectProject
멀티캠퍼스 2조 ArtConnect 프로젝트 (미술관 홈페이지)


## 서비스 핵심 기능
```
회원: 로그인 | 회원가입 | 이메일 인증 | 
마이페이지: 좋아요 조회 | 예약목록 조회
관리자페이지 : 회원목록 조회 | 메일 보내기
```
